# Laravel Ride Booking API

## Installation

1. Clone repo
2. composer install
3. cp .env.example .env
4. Configure DB
5. php artisan migrate
6. php artisan serve

## APIs
- POST /api/rides
- POST /api/rides/{id}/approve
- POST /api/drivers/location
- GET /api/drivers/rides

## Admin Panel
- /admin/rides
